# Oficina de Lógica de Programação com Pseudocódigo

Projeto de extensão desenvolvido no curso de Ciência da Computação – UNIP.

## Objetivo
Ensinar lógica de programação para iniciantes da comunidade externa utilizando pseudocódigo como ferramenta introdutória.

## Público-alvo
Estudantes do ensino médio e iniciantes em programação.

## Conteúdo
- Algoritmos
- Pseudocódigo
- Entrada e saída de dados
- Estruturas condicionais
- Laços de repetição
- Vetores
- Exercícios práticos resolvidos

## Acesso
Material gratuito e aberto à comunidade.
