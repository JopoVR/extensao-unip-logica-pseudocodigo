Exercício 3 – Maior e menor número

INÍCIO
    Leia num1
    Leia num2
    Leia num3

    maior ← num1
    menor ← num1

    SE num2 > maior ENTÃO
        maior ← num2
    FIMSE

    SE num3 > maior ENTÃO
        maior ← num3
    FIMSE

    SE num2 < menor ENTÃO
        menor ← num2
    FIMSE

    SE num3 < menor ENTÃO
        menor ← num3
    FIMSE

    Escreva "Maior número: ", maior
    Escreva "Menor número: ", menor
FIM
